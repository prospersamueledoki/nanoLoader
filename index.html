🚀 NanoLoader

A lightweight CSS + JS loader library with 15+ styles, size options, and color customization.

⚡ Getting Started

Include the CSS and JS files in your project:

<link rel="stylesheet" href="nanoLoader.css">
<script src="nanoLoader.js"></script>

⚙️ Customization

You can change loader appearance using:

  • Colors: Add classes like nl-color-red, nl-color-blue, or define your own with style="color:#ff00ff".
  • Sizes: Use nl-size-sm, nl-size-md, nl-size-lg, or define a custom size with JavaScript: document.getElementById("myLoader").dataset.size = "120px";
  • Dynamic Control: You can show or hide loaders in JS, or update color dynamically: document.getElementById("myLoader").style.color = "cyan";

📚 Examples

Example usage in a page where you want a loader while fetching data:

<div id="loader" class="nanoLoader nl-ripple nl-color-green"></div>
<script>
  // Show loader
  document.getElementById("loader").style.display = "block";
  // Simulate fetch
  setTimeout(() => {
    document.getElementById("loader").style.display = "none";
  }, 3000);
</script>
